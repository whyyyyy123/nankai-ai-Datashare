# 期末资料整理分享
## 1、项目性质
无偿的资料整理与分享活动，实现资料跨年级传播
## 2、项目目的
打破复习资料信息差，让大多数同学能够轻松得到往年复习资料，花费更少的时间来完成无意义的重复劳动，从而将更多的时间用于对自身发展有利的活动中去。
## 3、获取资料前应认同
所有的资料都是学长学姐们无偿整理，并且经过了一定的审核和补充的，我们只将资料用于复习和交流，不进行商业性质的买卖。
## 4、加入我们
项目为动态更新和实时调整的，所以每年都需要新的成员加入，我们需要您有资料收集和整理能力、媒体宣传能力、项目管理能力（其中一种即可）\
我们的分工暂时有：资料整理者、宣传者、项目管理者（暂时由项目发起人以及大四同学担任），每一种分工只是按照各自擅长的技能划分。\
加入我们后会按照技能进行分工，然后开展各自的工作，最后将我们的工作无偿贡献出来供全部同学使用和学习，我们希望您也是一个具有开源精神的分享者。\
目前我们缺少：24级、25级进行资料整理的同学(整理大一和大二相关资料)，以及不限年级的项目推广同学。
## 5、说明
再次强调我们的项目不收取任何费用，任何商业活动或者收费活动均不是我们项目成员承办，也不应该用以我们项目的名义。\
整个项目的开展过程中，不应涉及政治、暴力、反社会、黄色等违反道德和法律的信息，项目的管理者将会严格审核和管理。
## 6、资料更新方式
前期主要由指定同学整理，管理员审核统一上传\
后期会开放邮箱投稿方式，以及熟悉github的同学也可以通过Pull requests的方式直接投稿，管理员将会在一周之内进行处理。\
对项目有任何建议和想法，可以发送Issues进行讨论，合适的建议将会被采纳。\
署名问题：处理一般投稿时，默认不进行署名，如果有署名需求，请在邮箱或者Pr说明，并且自行提供README文档用于说明投稿具体内容，署名内容不限（可实名或其他能代表自己的昵称，但是不能含有不良信息）
